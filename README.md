## Minimalist | 极简

Minimalist | 极简是一款简洁的个人主页静态模板

### 功能

* 信息卡片（ 展示基本信息
* 搜索卡片（ 快速进行搜索
* 常用站点（ 展示您经常访问的站点
* 右键菜单（ 简单的自定义右键菜单
